## üÜò Troubleshooting

**Images not showing?**
- Check file path is correct
- Make sure images folder exists
- Verify image file names match HTML
- Try refreshing the browser (Ctrl+F5)

**Text hard to read on purple?**
- All text should now be white
- If you see black text, let me know which section

**FAQ not opening?**
- Make sure `script.js` is in the same folder as `index.html`
- Check browser console for errors (F12 key)

**Mobile menu not working?**
- Verify `script.js` is loaded
- Resize browser to less than 768px wide
- Look for hamburger icon (3 lines)
